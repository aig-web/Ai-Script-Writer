1. HOOK FORMULAS:
- [Problem] + [Unexpected Solution]
- [Common Belief] + [Contradiction]
- [Big Number] + [Surprise Element]
- [Authority Figure] + [Unexpected Action]
- [Popular Brand/Person] + [Controversy]
- [Common Activity] + [Hidden Risk/Reward]

2. PATTERN INTERRUPT STRUCTURE:
0-3s: Hook/Pattern Interrupt
3-5s: Context/Setup
5-10s: Problem Statement
10-15s: Solution/Action
15-25s: Development
25-35s: Twist/Revelation
35-45s: Resolution
45-50s: Call to Action

3. TENSION BUILDING PHRASES:
- "But here's where it gets interesting..."
- "Plot twist:"
- "What happened next was unexpected..."
- "That's when everything changed..."
- "Here's the part most people miss..."
- "But wait, there's more..."

4. RETENTION TRIGGERS:
- Step-by-step breakdowns
- Progress markers (1,2,3)
- Mid-video hooks
- Visual transition cues
- Countdowns
- Time-based urgency

5. ENGAGEMENT COMMANDS:
- Direct questions
- Controversial topics
- "Comment below if..."
- "Save this for later"
- "Share if you agree"
- Follow hooks for sequels

6. KEY SCRIPT ELEMENTS:
- Use present tense
- Short, punchy sentences
- Pattern interrupts every 3-4 seconds
- One main idea per text block
- Clear cause-and-effect relationships
- Emotional triggers

- Resolution
